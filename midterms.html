Term Reflection

For our Midterms we learned on how to use Spacy Library, so Spacy library is a free Natural Language Processing open library in python it is also designed, it is designed to make Natural Language Processing and can handle large amounts of texts for processing and understanding. Installing Spacy is easy and you only need to put this line of code:

!pip install spacy

Also there are some features presented in the Spacy are Tokenization, Part-of-Speech tagging (POS), Lemmatization, Named Entity Recognition (NER) and a lot more in their documentation. Spacy also has some models that you can use the en_core_web_sm which means small, the en_core_web_md which means midium and the en_core_web_lg means large, the sizes of the model varies in the performance, like for the small model it is fast and more resource efficient but is limited on the semantic understanding, for the medium size, it is more balanced and good for some general NLP tasks, and the Large size, it is more accurate and powerful and more advanced NLP tasks but it takes a lot of resoucres.

Also in the Spacy pipeline we can also try to customize on what should we can also specify what pipelines we can only use for example:

pipeline = ["tagger", "parser", "ner"]

In the Midterms we also learned about NLP Techniques like the Named Entity Recognition, which is useful for the nameing and locating of the entites in the text, this is used for categorization and segmentation for example in customer questions and quesries. Also Sentiment Analysis is the is a scale positive, negative, and neutral, it is mainly used for reviews from the customer and surveys.

I also learned how to make the performance of the models with testing the model using metrics, which are accuracy, precision, and recall, and the F1-score. The accuracy provides a straightforward approach where, in it measures the output correctly predicted out of total outputs in the dataset. The precision measures the correctly positive predictions while the recall measures how many are actual positive predictions. The F1-score is the harmonic mean of precision and recall, balancing false positives with false negatives. This allowed me to interpret these metrics more meaningfully when determining what changes the model needed to its reliability and accuracy.

One of the most exciting things about midterms was learning about BERT, the Bidirectional Encoder Representations from Transformers model. BERT has revolutionized the world of natural language processing because now models can be understood in both directions of a sentence-both ways, either left to right or right to left. This kind of bidirectional has made BERT quite effective in tasks like sentiment analysis, question answering, and text classification because it captures deeper contextual meaning from the text. I learned how it could take all the transformer-based models and transform them into giving improved accuracy and understanding nuances over language.
